📌 About the Project

This is a simple Music Website (One Page Scroll Design) built using only HTML and CSS.
It is a practice project created to improve frontend design skills and understand layout structuring design.

🚀 Features
🎨 Clean and modern UI design
📜 One page scroll layout
🧱 Organized sections (Overview, Music, Video, Gift Cards)
⚡ Lightweight static website (no JavaScript used)
🛠️ Technologies Used
HTML5
CSS3
📂 Project Type
Frontend Practice Project
Static Website
UI/UX Design Practice
🎯 Purpose

This project was created for learning and practice purposes to improve HTML and CSS skills, especially layout design and web development.
